# 🧮 Calculadora via terminal

Uma calculadora desenvolvida em **Python** para execução diretamente no terminal. O projeto foi criado com o objetivo de praticar conceitos fundamentais da linguagem, como estruturas condicionais, laços de repetição, funções, tratamento de exceções e interação com o usuário.

## 📌 Funcionalidades
1. ➕ Soma
2. ➖ Subtração
3. ✖️ Multiplicação
4. ➗ Divisão

- [x] 🔄 Menu interativo
- [x] ⚠️ Tratamento de erros (try / except)
- [x] 🚪 Opção para encerrar o programa
